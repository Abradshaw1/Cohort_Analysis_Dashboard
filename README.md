# Cohort Analysis Dashboard

EDA dashboard for comparing individual outcomes in cohorts from medical data.

## Goal

Build out cohort analysis capabilities across datasets and subgroup comparisons for medical research.

## Current Features

- Interactive PCA visualization with brushing
- Feature distribution analysis
- Subgroup comparison and summary statistics
- Built with React + D3.js

## Dataset

Currently uses the Framingham Heart Study dataset for demonstration.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
